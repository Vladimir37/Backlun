ToDo List Backlun

Main page - Backlun/front/todo/index.html.

Static folder - Backlun/front/todo/static/.

You can get the file from static folder at the address /src.

Example: Backlun/front/todo/static/js/build/main.js -> /src/js/build/main.js

Test it: Backlun/back/todo/_test_/request.test.js

Command required: npm install && npm run test

ToDo API

Params marked with asterisk (*) are required. The contents of the field "Response" is located in the field "body" in Server response object. When editing is necessary to send the contents of all fields - unsent field are cleared.

Method Address Params Response Description
GET /api/tasks/get_all/ - Array of Task objects Get the array of all ToDo tasks
GET /api/tasks/get_category/ id* int
 Array of Task objects Get the array of all ToDo tasks with desired category
POST /api/tasks/add/ title* string
text* string
category int 		Task object Create new ToDo task
POST /api/tasks/edit/ id* int
title string
text string
category int
status bool 		Task object Edit an existing ToDo task
POST /api/tasks/delete/ id* int - Delete an existing ToDo task
GET /api/categories/get_all/ - Array of Category objects Get the array of all tasks categories
POST /api/categories/add/ name* string
color* string 		Category object Create new task category
POST /api/categories/edit/ id* int
name string
color string 		Category object Edit an existing task category
POST /api/categories/delete/ id* int - Delete an existing task category

Standard objects

Server response:

status int
message string
body any

Status code: 0 - success, 1-9 - various errors, 10 - not found.

Task object:

ID int
Title string
Text string
Category int
Status bool

The field "category" should contain ID of the existing Category or 0.

Category object:

ID int
Name string
Color string

You can not delete a category, as long it contains the tasks
