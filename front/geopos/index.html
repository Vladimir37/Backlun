Geopos Backlun

Main page - Backlun/front/geopos/index.html.

Static folder - Backlun/front/geopos/static/.

You can get the file from static folder at the address /src.

Example: Backlun/front/geopos/static/js/build/main.js -> /src/js/build/main.js

Geopos API

Params marked with asterisk (*) are required. The contents of the field "Response" is located in the field "body" in Server response object.

Method Address Params Response Description
GET /api/getPoints - Array of Points objects Get the array of all geopos points
GET /api/getPointOnToken - A object of Point object Get a point of the token
POST /api/postPoint/ point* point
 status object Add a point in geostate
PUT /api/putDistance point* point
 distance you coordinats until checkPoint Put your point in geostate and return distance of checkPoint
GET /api/getCheckPoint - Point of Point objects Get the point of check
POST /api/postCheckPoint point* point - Update/Set new check point as primary goal
POST /api/postRndPoint point* point Point object post random point to geostate
GET /api/getRndPoint - Point of Point objects Get random point

Standard objects

Server response:

status int
message string
body any

Status code: 0 - success, 1-9 - various errors, 10 - not found.
