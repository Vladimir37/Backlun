geopos List Backlun

Main page - Backlun/front/geopos/index.html.

Static folder - Backlun/front/geopos/static/.

You can get the file from static folder at the address /src.

Example: Backlun/front/geopos/static/js/build/main.js -> /src/js/build/main.js

geopos API

Params marked with asterisk (*) are required. The contents of the field "Response" is located in the field "body" in Server response object. When editing is necessary to send the contents of all fields - unsent field are cleared.

Method Address Params Response Description
GET /getPoints - Array of Points objects Get the array of all geopos points
GET /getPointOnToken - A object of Point object Get a point of the token
POST /postPoint/ point* point
 status object Add a point in geostate
PUT /putDistance point* point
 distance you coordinats until checkPoint Put your point in geostate and return distance of checkPoint
GET /getCheckPoint - Point of Point objects Get the point of check
POST /postCheckPoint point* point - Update/Set new check point as primary goal
POST /postRndPoint point* point Point object post random point to geostate
GET /getRndPoint - Point of Point objects Get random point
