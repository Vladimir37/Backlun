Calendar API

Params marked with asterisk (*) are required. The contents of the field "Response" is located in the field "body" in Server response object. When editing is necessary to send the contents of all fields - unsent field are cleared. The field "category" should contain ID of the existing Category or 0. The Date type is string and looks like this: [year]-[month]-[day]T[hours]:[minutes]:[seconds].[milliseconds]Z. Example: 2017-03-10T23:13:52.207Z. You can get the time in this format through JavaScript method JSON.stringify.

Standard objects

Server response:

status int
message string
body any

Status code: 0 - success, 1-9 - various errors, 10 - not found.

Short event object:

ID int
Category Category object
Title string
Description string
Time date

Long event object:

ID int
Category Category object
Title string
Description string
StartTime date EndTime date

Category object:

ID int
Name string
Color string
